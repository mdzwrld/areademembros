# **App Name**: Lucro Discord Rápido

## Core Features:

- Secure User Authentication: Implement a simple login system with email-based auto-registration, global standard password assignment, and automated password delivery to the user's email. Includes session management and route protection for member and admin areas.
- Dynamic Member Video Content Display: Show a curated selection of embedded unlisted YouTube videos to authenticated members, each with a customizable title and an associated 'Necessary Links' text section. The content is managed via the admin panel.
- Administrator Video Content Management: A secure admin panel allowing authorized users to add new videos (specifying title, YouTube URL, and necessary links), edit existing video details, and remove videos from the platform.
- Administrator System Settings & User Overview: Enable administrators to update the global default password for new user registrations and to view a list of all currently registered member emails.
- AI-Powered Link Generation Tool: An administrative tool that leverages AI to assist in drafting and refining the 'Necessary Links' text for each video, using video context to suggest compelling and relevant calls to action or resources.
- Interactive Promotional Call-to-Actions (CTA): A dedicated, prominently displayed section at the bottom of the member's area, featuring interactive cards for recommended products with descriptions and clear 'BUY NOW' buttons.

## Style Guidelines:

- Primary color: A vibrant digital purple (#B980F2), selected for its energetic contrast on dark backgrounds and modern digital aesthetic.
- Background color: A very dark, subtle plum tone (#1C161E), derived from the primary hue but heavily desaturated, providing a sleek and modern canvas.
- Accent color: A luminous blue (#8AA1F7), analogous to the primary and chosen for its striking saturation and brightness to highlight calls to action and interactive elements.
- Headlines will use 'Space Grotesk', a sans-serif with a techy, contemporary feel. Body text and longer descriptions will use 'Inter', a neutral and highly readable grotesque-style sans-serif, ensuring clarity and professionalism.
- Utilize minimalist, modern line-art icons throughout the interface, with special attention to impactful, product-related icons within the CTA section to enhance visual communication.
- Implement a clean, simple, and responsive layout, optimized for mobile viewing. Content will be organized using 'cards' with rounded borders and subtle shadows for visual depth, especially for videos and CTA items. Key elements like titles will be centrally aligned for impact.
- Incorporate subtle hover effects on interactive elements like buttons and cards, along with smooth transitions for content loading and state changes, contributing to a fluid and polished user experience.
